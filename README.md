# Digital Workspace (Aurora)

A modern, full-featured digital workspace application built with React and Vite. Aurora provides teams with comprehensive collaboration tools including video conferencing, real-time chat, document sharing, password management, and team management capabilities.

### Core Features
- **Video Conference** - HD video calls with screen sharing, recording, and virtual backgrounds (supports up to 50 participants)
- **Team Chat** - Real-time messaging with file sharing, threads, and smart notifications
- **Document Collaboration** - Real-time document editing with comments and version history tracking
- **Password Manager** - Secure storage for passwords, meeting links, and login credentials
- **Team Management** - Comprehensive team organization and member management
- **Notifications** - Smart notification system to keep teams updated
- **Theme Support** - Light and dark mode support
- **Authentication** - Secure login and signup system with protected routes
